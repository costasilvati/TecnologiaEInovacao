Tecnologia e Inovação

Capítulo 7 Planilhas - parte 2

7.1 Objetivo

Neste material, vamos estruturar um arquivo .xlsx de gerenciamento financeiro com três planilhas:

  • Registros
  • Categorias
  • Carteiras

A meta é fazer com que, na planilha Registros, as colunas carteira e categoria aceitem apenas valores previamente cadastrados nas planilhas Carteiras e Categorias.

Além disso, o material apresenta duas implementações:

  • no Microsoft Excel;
  • no Google Planilhas.

7.2 Estrutura recomendada do arquivo

7.2.1 Planilha Registros

Colunas sugeridas:

  • data
  • desc
  • parcelas
  • carteira
  • categoria
  • Entrada/Saída

Exemplo:

data desc parcelas carteira categoria Entrada/Saída
01/04/2026 Salário 1 Nubank Salário Entrada
02/04/2026 Supermercado 1 Dinheiro Alimentação Saída
03/04/2026 Compra notebook 10 Cartão Tecnologia Saída

7.2.2 Planilha Categorias

Colunas:

  • id
  • nome

Exemplo:

id nome
1 Alimentação
2 Transporte
3 Moradia
4 Salário
5 Tecnologia

7.2.3 Planilha Carteiras

Colunas:

  • id
  • nome

Exemplo:

id nome
1 Dinheiro
2 Nubank
3 Inter
4 Cartão

Ideia central da solução

A lógica é simples:

  1. cadastrar as categorias e carteiras em abas auxiliares;
  2. transformar essas listas em fontes confiáveis de dados;
  3. aplicar validação de dados na aba Registros;
  4. impedir que o usuário digite valores fora das listas permitidas.

Na prática, isso cria listas suspensas nas colunas carteira e categoria.

7.3 Como fazer no Excel

Visão geral

No Excel, a validação de dados permite restringir o conteúdo de uma célula e criar listas suspensas. A Microsoft orienta usar Dados > Validação de Dados e, para listas armazenadas em outra planilha, recomenda definir um nome para o intervalo da lista. Isso torna a solução mais estável e mais fácil de manter.

Passo um: organizar as listas auxiliares

Na planilha Categorias, preencha a coluna nome com os nomes das categorias.

Na planilha Carteiras, preencha a coluna nome com os nomes das carteiras.

Sugestão prática:

  • deixe a linha 1 para os cabeçalhos;
  • coloque os dados a partir da linha 2;
  • evite linhas vazias no meio da lista.

Exemplo:

Categorias

A B
id nome
1 Alimentação
2 Transporte
3 Moradia
4 Salário

Carteiras

A B
id nome
1 Dinheiro
2 Nubank
3 Inter
4 Cartão

Passo dois: criar intervalos nomeados

Esse é o ponto mais importante no Excel.

Em vez de tentar apontar a validação diretamente para outra planilha, crie um nome para cada lista.

Intervalo para categorias

  1. Vá até a planilha Categorias.
  2. Selecione apenas os nomes das categorias, por exemplo: B2:B100.
  3. Acesse a guia Fórmulas.
  4. Clique em Definir Nome.
  5. Use o nome:
lista_categorias
  1. Confirme.

Intervalo para carteiras

  1. Vá até a planilha Carteiras.
  2. Selecione os nomes das carteiras, por exemplo: B2:B100.
  3. Acesse Fórmulas > Definir Nome.
  4. Use o nome:
lista_carteiras
  1. Confirme.

Passo três: aplicar validação na planilha Registros

Agora vá para a planilha Registros.

Validar a coluna categoria

Supondo que a coluna categoria seja a coluna E:

  1. selecione o intervalo onde os usuários irão digitar, por exemplo E2:E500;
  2. acesse Dados > Validação de Dados;
  3. em Permitir, escolha Lista;
  4. em Fonte, digite:
=lista_categorias
  1. confirme.

Validar a coluna carteira

Supondo que a coluna carteira seja a coluna D:

  1. selecione D2:D500;
  2. acesse Dados > Validação de Dados;
  3. escolha Lista;
  4. em Fonte, digite:
=lista_carteiras
  1. confirme.

Passo quatro: configurar a mensagem de erro

No Excel, você pode personalizar a mensagem mostrada quando alguém tentar digitar um valor inválido.

Na janela de validação:

  1. abra a aba Alerta de Erro;
  2. mantenha a opção para bloquear entradas inválidas;
  3. use um título, por exemplo:
Valor inválido
  1. use uma mensagem como:
Selecione um item da lista cadastrada.

Isso melhora a usabilidade da planilha.

Observação importante sobre atualização das listas no Excel

Se você cadastrou um intervalo fixo como B2:B100, a validação continuará funcionando bem enquanto os novos valores forem adicionados dentro desse limite.

Se quiser uma solução mais robusta, há duas opções:

  • definir um intervalo suficientemente grande, como B2:B1000;
  • transformar a base em tabela e depois manter um intervalo nomeado bem planejado.

Para um material didático e para planilhas administrativas, o intervalo fixo amplo costuma ser a solução mais simples.

7.4 Como fazer no Google Planilhas

Visão geral

No Google Planilhas, a lógica é semelhante, mas a configuração costuma ser mais direta. A ajuda oficial permite criar uma lista suspensa a partir de um intervalo e informa que, quando o conteúdo do intervalo-fonte muda, a lista é atualizada automaticamente.

Passo um: preencher as abas auxiliares

Repita a mesma organização:

Categorias

A B
id nome
1 Alimentação
2 Transporte
3 Moradia
4 Salário

Carteiras

A B
id nome
1 Dinheiro
2 Nubank
3 Inter
4 Cartão

Passo dois: validar a coluna categoria

Na aba Registros:

  1. selecione as células da coluna categoria, por exemplo E2:E500;
  2. clique em Dados > Validação de dados;
  3. clique em Adicionar regra;
  4. em Critérios, escolha Lista suspensa a partir de um intervalo;
  5. informe o intervalo:
Categorias!B2:B
  1. marque a opção para rejeitar entrada inválida, se desejar impedir digitação fora da lista;
  2. clique em Concluído.

Passo três: validar a coluna carteira

Na aba Registros:

  1. selecione D2:D500;
  2. vá em Dados > Validação de dados;
  3. crie uma nova regra;
  4. escolha Lista suspensa a partir de um intervalo;
  5. informe:
Carteiras!B2:B
  1. conclua.

Vantagem prática no Google Planilhas

No Google Planilhas, o uso de intervalos abertos como B2:B é muito útil. Assim, quando novas categorias ou carteiras são adicionadas, elas passam a aparecer automaticamente na lista, sem necessidade de editar a regra manualmente.

Como validar também a coluna Entrada/Saída

Como a coluna Entrada/Saída possui poucas opções e elas quase nunca mudam, você pode criar uma validação simples diretamente na própria regra.

7.4.1 No Excel

  1. selecione a coluna correspondente, por exemplo F2:F500;
  2. vá em Dados > Validação de Dados;
  3. escolha Lista;
  4. em Fonte, digite:
Entrada,Saída

Observação: dependendo da configuração regional do Excel, o separador pode variar. Em muitos casos, especialmente no Brasil, o separador correto pode ser ; em vez de ,.

7.4.2 No Google Planilhas

  1. selecione F2:F500;
  2. vá em Dados > Validação de dados;
  3. escolha Lista suspensa;
  4. adicione os itens:
    • Entrada
    • Saída

7.5 Boas práticas para esse tipo de arquivo

7.5.1 Padronizar nomes das abas

Use exatamente estes nomes:

  • Registros
  • Categorias
  • Carteiras

Isso evita erros nas referências.

7.5.2 Não permitir duplicidades desnecessárias

Em Categorias e Carteiras, evite repetir nomes.

Exemplo inadequado:

id nome
1 Alimentação
2 Alimentação

Isso gera listas com opções repetidas e dificulta análises posteriores.

7.5.3 Manter o cadastro como base mestra

As planilhas Categorias e Carteiras devem funcionar como tabelas de cadastro. O ideal é que os usuários não criem categorias novas diretamente em Registros.

7.5.4 Preparar pensando em análise de dados

Essa modelagem ajuda muito em relatórios futuros, porque:

  • reduz erros de digitação;
  • evita categorias escritas de formas diferentes;
  • facilita filtros e tabelas dinâmicas;
  • melhora a qualidade dos dados para dashboards.

Exemplo de problema evitado:

  • Alimentação
  • alimentação
  • Alimentacao
  • Alim.

Sem validação, o sistema trata esses valores como categorias diferentes.

7.6 Exemplo de fluxo de uso

7.6.1 Cadastro de categorias

Na aba Categorias:

id nome
1 Alimentação
2 Transporte
3 Moradia
4 Saúde
5 Lazer
6 Salário

7.6.2 Cadastro de carteiras

Na aba Carteiras:

id nome
1 Dinheiro
2 Conta Itaú
3 Cartão XP
4 Nubank

7.6.3 Lançamentos em Registros

data desc parcelas carteira categoria Entrada/Saída
05/04/2026 Mercado 1 Nubank Alimentação Saída
06/04/2026 Uber 1 Dinheiro Transporte Saída
07/04/2026 Salário empresa 1 Conta Itaú Salário Entrada

Observe que carteira e categoria deixam de ser texto livre e passam a seguir um cadastro controlado.

7.7 Quando usar ID e quando usar nome

Para entrada manual no dia a dia, é melhor que a lista suspensa mostre o nome, e não o id.

O id é útil para:

  • organização interna;
  • possíveis integrações futuras;
  • relacionamentos em modelos mais avançados.

Mas, para a aba Registros, o mais amigável é selecionar diretamente nomes como Nubank, Alimentação e Transporte.

7.8 Resumo comparativo

Aspecto Excel Google Planilhas
Lista suspensa Sim Sim
Validação por intervalo em outra aba Melhor via intervalo nomeado Direto por intervalo
Atualização automática da lista Depende da forma como o intervalo foi criado Mais simples com intervalo aberto
Mensagem de erro Sim Sim

7.9 Conclusão

Para controlar as colunas carteira e categoria com base em cadastros prévios, a estratégia correta é usar validação de dados com listas suspensas.

No Excel, a forma mais segura é:

  • criar os cadastros nas abas auxiliares;
  • definir intervalos nomeados;
  • usar esses nomes como fonte da validação.

No Google Planilhas, a forma mais prática é:

  • criar os cadastros nas abas auxiliares;
  • usar a opção Lista suspensa a partir de um intervalo apontando para as colunas de nomes.

Essa estrutura melhora a consistência dos dados e prepara o arquivo para análises futuras, como tabelas dinâmicas, gráficos e dashboards.
